GameWebServer
----------------------------------------------------------------
   Игровой сервер с Многопоточностью, Системой сообщений, Ресурсной системой (XML, возможно VFS),
Системой авторизации, Database-сервисом (пул соединений\либо одно статичное соединение),
Механикой, JavaScript, Сервером Jetty. БД - MySqL.
   Работа с БД будет возможна через 2 реализации - JDBC и Hibernate.
   Данные между потоками будут передаваться самописной системой сообщений, лучшей чем RabbitMQ.
   Механика в ближайшее время будет простейшая.
   Страница /admin существует для того, чтобы завершить работу сервера Jetty. Для этого пользователь должен быть авторизован, иметь свойство admin=1 (нигде не проставляется, кроме как в БД).
   Сервер готов примерно наполовину.


maven
-----------------------------------------------------------------
Простое приложение под Wildfly + Maven + Servlet + EJB + MySQL
Deploy проходит и работает, если из war-файла выбросить dom4j.jar (вставляется Hibernate)
Также для успешной работы добавил конфиг в standalone.xml сервера:

<subsystem xmlns="urn:jboss:domain:ee:4.0">

   <global-modules>

      <module name="org.dom4j" slot="main"/>

   </global-modules>
